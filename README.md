# Backend
```bash
cd backend // navigate to backend
python3 -m venv venv // Create a virtual environment inside venv folder
source venv/bin/activate // Activate the virtual environment 
pip install -r requirements.txt // Install all the packages from requirements.txt 
uvicorn main:app --reload --host 0.0.0.0 --post 8000 // Start the server using uvicorn
```

# Frontend
```bash
cd frontend
npm run dev
```
